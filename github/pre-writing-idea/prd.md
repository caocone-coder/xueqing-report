## 基础指令
/frontend-design
生成**写作思路引导页面**高保真可交互原型，严格遵循流程、模块、交互与设计规范，无emoji，逻辑闭环，可直接运行演示。

---

## 核心写作流程（固定线性流程，不可随意跳转）
审题解读 → 选材引导 → 立意确认 → 框架搭建 → 提纲确认 → 写作素材推送

---

## 分模块详细功能与交互要求
### 1. 审题解读模块
- 页面标题：明确题目要求
- 固定题目：我的植物朋友
- 核心内容：
  1. 展示完整作文题目与写作要求
  2. AI自动总结核心考点
  3. AI自动总结写作注意事项
  4. 输出标准审题结论
- 交互规则： AI 老师3D 人物插画模拟语音交互与逐条打字展示完毕后**引导用户点击跳转**至选材引导模块
- 页面布局：参考/Users/a1-6/github/pre-writing-idea/demo/20260325-185026.jpeg

### 2. 选材引导模块
- 页面标题：确定写作内容
- 核心内容：AI 老师3D 人物插画模拟语音交互与逐条打字发起层层递进式提问，引导学生详细描述写作素材，鼓励具体回答，若为离题请引导用户再次拟好题目，如果学生无法拟题，AI 自动提供3 个题目供选择
- 提问方向：植物名称、外形特点、生长习性、相处故事、喜爱原因
- 交互规则：学生完成回答后，点击确认按钮进入下一模块

### 3. 立意确认模块
- 页面标题：确定写作中心思想
- 核心内容：AI 老师3D 人物插画模拟语音交互与逐条打字引导学生提炼文章核心中心思想，为核心必选环节
- 兜底规则：学生无法自主提炼时，AI自动提供2-3个合理立意方案供选择
- 交互规则：确认立意后点击按钮进入下一模块

### 4. 框架搭建模块
- 页面标题：确定写作提纲
- 固定引导顺序：开头 → 中间 → 结尾
- 交互规则：AI 老师3D 人物插画模拟语音交互与逐条打字分段引导、分段确认，确认一段后自动进入下一段
- 联动功能：页面内置思维导图，每段内容确认后**实时同步更新对应节点**
- 完成规则：三段全部确认完成后，自动进入提纲确认模块

### 5. 提纲确认模块
- 页面标题：写作提纲最终确认
- AI主动提问：是否确认使用当前写作提纲？
- 操作选项1：确认提纲
  - 跳转至写作总结页面：写作要求、写作内容、提纲、同步 AI 课
  - 包括跳转功能：AI思路成文、作文批改、写作素材
- 操作选项2：修改提纲
  - AI询问修改部分
  - 重新引导编辑对应内容
  - 同步更新思维导图
  - 重新发起提纲确认
- 页面布局：参考/Users/a1-6/github/pre-writing-idea/demo/20260325-185035.jpeg

### 6. 写作素材模块
- 触发条件：确认提纲后自动加载
- 展示内容：与题目《我的植物朋友》匹配的
  1. 优秀范文
  2. 好词好句
  3. 写作素材
  4. 常用句式
- 页面布局：参考/Users/a1-6/github/pre-writing-idea/demo/20260325-185044.jpeg

## 设计规范
1. 页面布局：教育类简洁工具风，包含顶部标题栏、流程步骤、主内容区、操作按钮区。全屏移动端页面，不要桌面端的手机壳包裹
2. 流程导航：展示全部步骤，**高亮标注当前步骤**
3. 内容区：清晰展示AI提示、学生输入区（支持文本和语音输入）、提纲预览、思维导图展示区
4. 交互规范：所有操作提供明确按钮与文字提示，无歧义
5. 视觉要求：无任何emoji表情
6. 适配场景：面向中小学生，操作简单、引导清晰、逻辑直观
